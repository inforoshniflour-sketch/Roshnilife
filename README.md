# ROSHNI Multi-Grain Flour - E-commerce Website

Premium multi-grain flour e-commerce website with complete checkout system.

## Features

- ✅ Complete checkout system
- ✅ Order tracking with unique order numbers (ROSH-2026-XXXX)
- ✅ Shipping calculator (150-250 PKR based on city)
- ✅ Order confirmation page
- ✅ Customer details form
- ✅ Payment method selection (COD/Card)
- ✅ WhatsApp integration ready
- ✅ Email notifications ready
